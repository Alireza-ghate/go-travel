# GoTravel ✈️

A modern and visually engaging travel landing page designed to inspire users to explore destinations and discover new travel experiences.
Go Travel serves as the landing page for a future travel platform. It showcases destinations, travel-related content, and promotional sections while providing a clean and intuitive interface across all devices.

## 🚀 Features

- Modern and responsive design
- Mobile-first user experience
- Smooth page animations and transitions
- Destination showcase sections
- Reusable and scalable component architecture
- Fast development workflow powered by Vite
- Type-safe codebase with TypeScript
- Optimized performance and loading experience

## 🛠️ Tech Stack

### Frontend

React 19
TypeScript
Vite

### Styling

Tailwind CSS v4

### Animations

Motion

### Data Fetching

TanStack React Query

### Backend Services

Supabase

## ⚙️ Getting Started

### Clone the repository

git clone https://github.com/Alireza-ghate/go-travel.git

### Navigate to the project directory

cd go-travel

### Install dependencies

npm install

### Run the development server

npm run dev
